# ADHD Planner demo pack

This folder is for demo preparation, not production logic.

Use it to show the app without exposing real personal tasks.

Files:
- `DEMO_SCRIPT_RU.md` — short 2-3 minute demo flow.
- `DEMO_DATA_RU.md` — safe fake tasks for a clean demo account.
- `DEMO_CHECKLIST_RU.md` — what to check before showing the app.

Rule:
- Do not demo with real private tasks.
- Do not show raw internal audits to users/investors.
- Show the product through a prepared story.
